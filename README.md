# popsnap - キッズアートメーカー

お気に入りの写真をポップなアートに変身させるWebアプリです。

## 機能
- 写真のアップロード（カメラ/ライブラリ）
- 12種類のユニークなアートスタイル生成
- Gemini APIを使用したAI画像加工
- 日本語UI対応
- シェア機能

## 開発環境
- React + Vite
- TypeScript
- Tailwind CSS
- Google Gemini API

## 注意事項
APIキーは環境変数 `API_KEY` から読み込まれます。
